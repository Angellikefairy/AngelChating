## 功能
1. 添加好友、添加群组、私聊、群聊
2. 支持图片发送、默认表情选取
3. Web网页截图（由于Canvas的限制，截图会获取到隐含的像素数据）
4. 桌面消息通知（使用H5的新特性Notification）


## 安装与运行

```sh
npm install

npm run start
```
